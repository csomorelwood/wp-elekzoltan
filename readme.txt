== Description ==

Ez a plugin aktiváláskor becsokizza az oldalt és a teljes képernyőt beborítja Zolibácsi képeivel, mindezt zenei aláfestés közepette.
Kárt nem okoz, nem nyúlkál bele semmibe, csak letilja a kattintásokat, meg feldob egy modalt.
Állítgatni is lehet a csoki mennyiségét, de ahhoz csak inspectorral férsz hozzá.
Kikapcsolni is inspectorral tudod. Have fun :)

EN: The plugin disables the pointer events, and just pops a modal.
You can easily modify the settings and disable it in inspector mode.
